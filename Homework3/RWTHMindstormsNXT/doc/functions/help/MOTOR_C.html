MOTOR_C

Symbolic constant MOTOR_C (returns 2)

Contents

  • Syntax
  • Description
  • Example
  • See also
  • Signature

Syntax

portC = MOTOR_C()

Description

portC = MOTOR_C() returns 2 as the value portC.

Example

 portC = MOTOR_C()
 %result: >> portC = 2
