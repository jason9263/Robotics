SENSOR_1

Symbolic constant SENSOR_1 (returns 0)

Contents

  • Syntax
  • Description
  • Example
  • See also
  • Signature

Syntax

port1 = SENSOR_1()

Description

port1 = SENSOR_1() returns 0 as the value port1.

Example

  port1 = SENSOR_1()
  %result: >> port1 = 0
