RWTH - Mindstorms NXT Toolbox

GUIs

  • GUI_WatchSensor.m - Tool to read the NXT sensors and monitor the data values.
  • GUI_WatchMotorState.m Tool to read the NXT motors and monitor the data values.

Tools

  • ToolboxBenchmark.m - Measures performance and latencies for sensor and motor control commands

  • ToolboxTest.m - Performes certain automated tests and integrity checks or measures motor control precision

  • ToolboxUpdate.m - Shows the local installed toolbox version and gets remote update information from the toolbox SVN server.
